# Display Presets

Display presets provide fast, repeatable visual styles for figure preparation and analysis.

---

## Canvas presets

The publication canvas exposes three global presets:

| Preset | Description |
|---|---|
| Clean | Minimal display clutter |
| Analysis | More analytical overlays and aids |
| Publication | Journal-style presentation with reduced clutter |

Manual changes return the canvas to **Custom** state.

See [Publication Canvas](canvas.md).

---

## Why presets matter

Presets let you switch quickly between:

- an exploratory analysis view
- a cleaner presentation view
- a publication-style figure layout

This is faster and more reliable than toggling every overlay one by one.

---

## Related workflows

The profile measurement tools also include publication-oriented sizing work, and popup/style workflows allow one window's style to be copied to others.

That makes presets part of a broader figure-preparation pipeline rather than a canvas-only convenience.
