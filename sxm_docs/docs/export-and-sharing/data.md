# CSV & Data Export

SXM Viewer can export numerical data as well as rendered figures.

---

## Typical data exports

Depending on the tool, exported data can include:

- profile or spectroscopy values
- comparison outputs
- channel data for further analysis elsewhere

The spectroscopy popup also exposes direct data-copy actions for trace content, and the A/B comparison workflow includes CSV export alongside image export.

---

## When to use data export

Use CSV or data-copy workflows when you need to:

- replot in another tool
- perform custom fitting outside the GUI
- archive the numerical values behind a figure
- compare several exported traces in a spreadsheet or script

---

## Related workflows

- spectroscopy windows can copy trace data directly
- comparison workflows can export analysis outputs
- image export and data export are separate, so visual styling does not overwrite the underlying numbers
