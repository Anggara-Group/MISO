# Export to PowerPoint

SXM Viewer can export figures for use in PowerPoint while preserving the source image proportions.

---

## What is exported

PowerPoint export is intended for presentation-ready images and layouts, including canvas-based figure composition.

Recent project changes explicitly fixed the export path so source aspect ratios are preserved instead of forcing every image into the same fixed rectangle. In practice, that means square scans stay square.

---

## Typical workflow

Use PowerPoint export when you want:

- slide-ready figures
- image layouts assembled on the publication canvas
- captions or titles adapted for presentation use

Single-view and canvas workflows can both feed presentation output.

---

## Notes on appearance

For some single-view exports, the top title is hidden temporarily when a bottom caption is being added, which avoids duplicate titling in the final slide asset.
